gateway简介

Spring Cloud Gateway是spring团队基于netty重写的API Gateway组件，相对于Zuul性能较好，其不仅提供统一的路由方式，并且基于Filter链的方式提供了网关基本的功能，例如：安全，监控/埋点，和限流等。

gateway工作原理

gateway功能

  1. 协议转换，路由转发
  2. 流量聚合，对流量进行监控，日志输出
  3. 作为整个系统的前端工程，对流量进行控制，有限流的作用
  4. 作为系统的前端边界，外部流量只能通过网关才能访问系统
  5. 可以在网关层做权限的判断
  6. 可以在网关层做缓存
